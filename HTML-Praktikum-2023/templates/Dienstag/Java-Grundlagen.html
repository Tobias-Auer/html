public class Main {

  }

Danach haben wir uns mit Variablen und Datentypen auseinander gesetzt und uns die Grundrechenarten in Java angeschaut. System.out.println("Hello World");

int firstInt = 41;

Am Ende der Stunde haben noch gelernt wie wir Userinput über die Konsole empfangen können. Mit unserem neu erlenten Wissen waren wir in der Lage einen Taschenrechner zu implementieren. Der Code dafür war der folgende: import java.util.InputMismatchException;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scannerObj = new Scanner(System.in);
        while (true) {
            try {
                System.out.print("Enter first digit: ");
                double firstDigit = scannerObj.nextDouble();
                System.out.print("Enter operator: ");
                String operator = scannerObj.next();
                System.out.print("Enter second digit: ");
                double secondDigit = scannerObj.nextDouble();
                switch (operator) {
                    case "+" -> {
                        System.out.println(firstDigit + secondDigit);
                    }
                    case "-" -> {
                        System.out.println(firstDigit - secondDigit);
                    }
                    case "*" -> {
                        System.out.println(firstDigit * secondDigit);
                    }
                    case "/" -> {
                        try {
                            System.out.println(firstDigit / secondDigit);
                        } catch (ArithmeticException e) {
                            System.out.println("Man kan NICHT durch null teilen!!");
                        }
                    }
                    default -> System.out.println("Fehlerhafter operator!!!");
                }
            } catch (InputMismatchException e) {
                System.out.println("Bitte gebe richtige Werte ein!!");
            }

        }
    }
}
